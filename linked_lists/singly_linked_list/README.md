# Singly-Linked List

## Old implementation of a linked list of integers

## Operations

The following operations are defined in in [`ll.c`](ll.c)
and can be made available by including the [`ll.h`](ll.h) header:
- `append`
- `clear`
- `copy`
- `count`
- `extend`
- `insert`
- `new`
- `pop`
- `print`
- `remove`
- `reverse`
